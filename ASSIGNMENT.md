# Purpose

This assignment will help you set up or practice using GitHub, write and share simple scripts in two programming languages, and begin learning from your classmates’ coding style and creativity. You will create a GitHub repository, add a personalized “Hello World” script in Python and JavaScript, share your GitHub handle with the class, and respond to one classmate’s work.

## Instructions

### Create a GitHub repository

Create a new GitHub repository for this assignment. You may name it something clear, such as hello-world-ai-engineering, course-introduction, or another appropriate name of your choice.

### Create a Python “Hello World” script

Add a Python file to your repository. Name the file something clear, such as `hello.py`.

Your script should print a “Hello World” message, but it should also include a personal flare. For example, you might include your name, hometown, professional interest, favorite technology topic, a fun fact, or a short message about why you are taking this course.

Example idea:

`print("Hello, World! My name is Alex, and I am excited to learn how AI engineering can support secure public-sector modernization.")`

### Create a JavaScript “Hello World” script

Add a JavaScript file to the same repository. Name the file something clear, such as `hello.js`.

This script should communicate the same general message as your Python script, but written in JavaScript.

Example idea:

`console.log("Hello, World! My name is Alex, and I am excited to learn how AI engineering can support secure public-sector modernization.");`

### Add a short `README` file

Include a brief `README.md` file in your repository. Your `README` should include:

- The purpose of the repository
- The names of the two script files
- A sentence explaining the personal flare you added
- Share your GitHub handle with the class

### In the Canvas discussion board for this assignment, post your GitHub handle and a link to your repository.

Your post should include:

- Your name
- Your GitHub handle
- A link to your repository
- One sentence about the personal flare you included in your scripts
- Review one classmate’s script

### Visit at least one classmate’s repository. Review their Python or JavaScript script and respond to their Canvas post.

In your response, comment specifically on the personal flare they added. You might mention what you learned about them, what made their message memorable, or how their script reflected their interests or personality.

## Submission

Submit your work by posting in the Canvas discussion board. Your post must include your GitHub handle and a working link to your repository.

### What to Submit

Your Canvas discussion post should include:

- Your name
- Your GitHub handle
- A link to your GitHub repository
- A brief note describing your personal flare
- You must also reply to at least one classmate’s post with a comment about the personal flare in their script.

## Completion Criteria

To complete this assignment, you must:

- Create a GitHub repository
- Add a Python “Hello World” script with personal flare
- Add a JavaScript “Hello World” script with similar personal flare
- Add a short `README` file
- Share your GitHub handle and repository link with the class
- Comment on one classmate’s script, specifically addressing their personal flare
